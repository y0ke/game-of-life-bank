# かんたん銀行役 🎯

人生ゲームやモノポリーなどボードゲームで紙のお金を配る手間を省き、スマートフォンでタップとスワイプだけで簡単にお金の管理ができるWebアプリケーションです。

## 🎮 デモ

GitHub Pagesでホスティングされています：
[https://y0ke.github.io/game-of-life-bank/](https://y0ke.github.io/game-of-life-bank/)

## ✨ 主な機能

### 🎲 ゲーム設定
- **プレイヤー登録**: 2〜8人までのプレイヤーを登録
- **通貨選択**: JPY (¥) または USD ($) を選択可能
- **初期所持金設定**: デフォルト5,000から自由に設定可能
- **カラー選択**: 人生ゲームの車の色から好きな色を選択可能

### 💰 銀行機能
- **入出金管理**: 銀行からの受取・銀行への支払いを簡単管理
- **プレイヤー間送金**: プレイヤー同士の送金機能
- **クイック金額ボタン**: よく使う金額をワンタップで入力
- **加算式入力**: 金額ボタンをタップするたびに加算

### 📊 管理機能
- **取引履歴**: すべての取引を時系列で記録・表示
- **取引の取り消し**: 直前の取引をアンドゥ可能
- **マイナス残高対応**: 借金状態を赤色で警告表示
- **データ永続化**: LocalStorageによる自動保存

### 🏆 ゲーム終了
- **最終順位表示**: 所持金順にランキング表示
- **統計情報**: 最高額、最低額、平均額、総資産を表示
- **優勝者のハイライト**: 1位のプレイヤーを特別表示

## 🚀 使い方

1. **ゲーム開始**
   - プレイヤー名を入力（2〜8人）
   - 通貨を選択（JPY/USD）
   - 初期所持金を設定

2. **ゲーム中**
   - プレイヤーカードをタップして選択
   - 取引タイプ（受取/支払）を選択
   - 金額ボタンをタップして金額を入力
   - 「実行」ボタンで取引を確定

3. **ゲーム終了**
   - 右上の「終了」ボタンをタップ
   - 最終結果を確認
   - 「新しいゲームを開始」で次のゲームへ

## 🛠️ 技術スタック

- **フレームワーク**: Next.js 14 (App Router)
- **言語**: TypeScript
- **スタイリング**: Tailwind CSS
- **データ保存**: LocalStorage
- **ホスティング**: GitHub Pages

## 📱 特徴

- **レスポンシブデザイン**: スマートフォン、タブレット、PCに対応
- **オフライン対応**: インターネット接続不要
- **PWA対応**: ホーム画面に追加可能
- **高速動作**: 静的HTMLとして配信

## 🔧 開発

### 必要要件
- Node.js 18以上
- npm または yarn

### セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# ビルド
npm run build

# 静的ファイルのプレビュー
cd out && python3 -m http.server 8080
```
