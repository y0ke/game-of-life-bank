# 人生ゲーム銀行アプリを作成してください

## 概要
人生ゲームで紙のお金を配る手間を省き、スマートフォンでタップとスワイプだけで簡単にお金の管理ができるWebアプリを作成してください。

## 技術仕様
- **スタンドアロンで動作するwebページベースのアプリケーション
- **データ保存**: LocalStorageを使用（サーバー不要）
- **ファイル構成**: index.html（すべて1ファイルに統合）
- **対応デバイス**: スマートフォン優先のレスポンシブデザイン
- **最終的にコードの書き出し後にstaticなhtmlとjavascriptになれば良いので、開発言語自体はtypescript,next.jsを使ったものとする

## 必須機能

### 1. ゲーム開始画面
- プレイヤー名の登録（2〜8人）
- 各プレイヤーに自動で色を割り当て（判別しやすい8色）
- 初期所持金の設定（デフォルト: 200,000円）
- 登録したプレイヤーの削除機能
- 「ゲーム開始」ボタン

### 2. メイン銀行画面
- **プレイヤーカード表示**
  - 横スクロール可能なカード形式
  - 選択中のプレイヤーをハイライト
  - 名前、色、現在の所持金を表示

- **取引タイプ選択**
  - 「銀行から受取」「銀行へ支払」の切り替えタブ

- **金額入力**
  - クイック金額ボタン（10,000、20,000、30,000、50,000、100,000、200,000円）
  - カスタム金額入力フィールド
  - 「実行」ボタンで取引確定

- **プレイヤー間送金**
  - 他のプレイヤーへの送金ボタン
  - 送金先を選択して金額を指定

### 3. 履歴画面
- すべての取引を時系列で表示
- 各取引に以下を表示:
  - 取引タイプのアイコン（収入/支出/送金）
  - プレイヤー名
  - 金額
  - 時刻

### 4. ゲーム終了機能
- 「ゲーム終了」ボタン
- 最終順位の表示（所持金順）
- 新しいゲームの開始オプション

## デザイン要件

### カラースキーム
```css
メイン色: #667eea（紫）
サブ色: #764ba2（深紫）
成功: #28a745（緑）
エラー: #dc3545（赤）
背景: #f8f9fa（薄グレー）

プレイヤー色（8色）:
#FF6B6B（赤）, #4ECDC4（青緑）, #95E77E（黄緑）, #FFA07A（オレンジ）
#87CEEB（空色）, #DDA0DD（紫）, #F0E68C（黄）, #FFB6C1（ピンク）
```

### UI/UX
- 角丸のモダンなデザイン
- 大きめのタップターゲット（最小44px）
- スムーズなアニメーション
- 分かりやすいアイコン使用

## 操作方法
- **プレイヤー選択**: カードをタップまたは左右スワイプ
- **金額入力**: ボタンタップまたはキーボード入力
- **取引実行**: 大きな実行ボタンをタップ
- **画面切り替え**: 下部のナビゲーションタブ

{

## 実装の優先順位
1. 基本的な画面遷移とレイアウト
2. プレイヤー登録機能
3. 銀行との入出金機能
4. LocalStorageへの保存
5. 取引履歴表示
6. プレイヤー間送金
7. アニメーションとビジュアル改善

## 追加考慮事項
- 誤操作防止（確認ダイアログ）
- 数値のフォーマット（カンマ区切り）
- 取引のアンドゥ機能（直前の取引のみ）
- ゲームデータのリセット機能

すべての機能を1つのHTMLファイルに実装し、スタイルはstyleタグ内に、JavaScriptはscriptタグ内に記述してください。外部ライブラリは使用せず、モダンなJavaScript（ES6+）で実装してください。
